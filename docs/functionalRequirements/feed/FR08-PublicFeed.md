## FR-08 — View Public Feed

## Description
A visitor should be able to view the most upvoted posts across all hobbies on the platform, without needing to log in.

## Actors
Visitor

## Pre-conditions
- The system has at least one post registered.

## Main flow
1. User navigates to the home page.
2. System displays the most upvoted posts across all hobbies, ordered by votes descending.

## Alternative flows

**AF-01 — No posts available**
At step 2, if the platform has no posts yet, the system displays a message encouraging the user to create an account and share their first post.

## Post-conditions
- The user can browse the most popular content on the platform.
