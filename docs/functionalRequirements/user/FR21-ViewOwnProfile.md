## FR-21 — View Own Profile

## Description
An authenticated user should be able to view their own profile, including their username, description, hobbies, and posts, with the option to edit their account.

## Actors
Authenticated user

## Pre-conditions
- The user is authenticated.

## Main flow
1. User navigates to their own profile page.
2. System displays the user's username, description, hobbies, and posts.
3. System displays an edit button to allow the user to edit their account information.

## Alternative flows

## Post-conditions
- The user's own profile is displayed with editing options.

## Requires
- FR-02 — Login
- FR-01 — Create Account
