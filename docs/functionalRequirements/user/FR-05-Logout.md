# FR-05 — Logout

## Description
An authenticated user should be able to terminate their current session.

## Actors
Authenticated user

## Pre-conditions
- The user is authenticated.

## Main flow
1. User selects the logout option.
2. System terminates the current session.
3. User is redirected to the login page.

## Alternative flows
—

## Post-conditions
- Session is terminated.
- User is redirected to the login page.

## Requires
FR-02 — Login

## Side effects
—
