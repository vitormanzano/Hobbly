## US-17 — Create Comment

## Priority: High

## User Story
As an authenticated user, I want to comment on a post or reply to a comment, so that I can participate in discussions within the community.

### Acceptance Criteria
- [ ] The user can submit a comment on any existing post.
- [ ] The user can reply to an existing comment; the reply is nested under the parent comment.
- [ ] The system rejects submission if the content is empty.
- [ ] If the post was deleted before submission, the system displays an error and does not create the comment.
- [ ] The new comment appears on the page without a full reload.
- [ ] The post's comment count is incremented.
