## US-18 — Edit Comment

## Priority: Medium

## User Story
As an authenticated user, I want to edit one of my comments, so that I can correct or improve what I wrote.

### Acceptance Criteria
- [ ] The edit button is only visible to the comment author.
- [ ] The edit input is pre-filled with the current comment content.
- [ ] The system rejects submission if the content is empty.
- [ ] If the content is identical to the original, the edit is discarded silently.
- [ ] If the comment was deleted before submission, the system displays an error.
- [ ] The updated comment appears on the page without a full reload with an "edited" indicator.
