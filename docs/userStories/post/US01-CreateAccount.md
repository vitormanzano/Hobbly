## US-01 — Create Account

## Priority: High

## User Story
As a visitor, I want to create an account, so that I can access the platform and interact with the community.

### Acceptance Criteria
- [ ] The visitor must provide a username, email, and password.
- [ ] The system must reject an email that is already registered.
- [ ] The system must reject a username that is already taken.
- [ ] The system must validate the provided data before creating the account.
- [ ] The account is created and the visitor is redirected to the login page.
