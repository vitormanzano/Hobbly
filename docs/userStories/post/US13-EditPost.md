## US-13 — Edit Post

## Priority: Medium

## User Story
As an authenticated user, I want to edit my post within one hour of creating it, so that I can fix mistakes or improve my content.

### Acceptance Criteria
- [ ] The edit button is only visible to the post author.
- [ ] The edit button is not displayed if more than one hour has passed since the post was created.
- [ ] The edit form is pre-filled with the current post data.
- [ ] The system validates the updated data before saving.
- [ ] After saving, the user is redirected to the updated post page.
