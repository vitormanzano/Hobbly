## US-15 — View Post

## Priority: High

## User Story
As a visitor, I want to view a post and its comments, so that I can read the community's content without needing an account.

### Acceptance Criteria
- [ ] The post page displays the title, description, type, hobby, author, and comments.
- [ ] If the post does not exist or has been deleted, the system displays a not-found message.
