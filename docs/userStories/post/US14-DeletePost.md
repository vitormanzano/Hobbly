## US-14 — Delete Post

## Priority: Medium

## User Story
As an authenticated user, I want to delete my post, so that I can remove content I no longer want to share.

### Acceptance Criteria
- [ ] The delete button is only visible to the post author.
- [ ] The system displays a confirmation dialog warning that the action cannot be undone.
- [ ] If the user cancels, the post remains unchanged.
- [ ] If the user confirms, the post and all its associated comments and votes are permanently deleted.
- [ ] After deletion, the user is redirected to the main page.
