## US-16 — Search Posts

## Priority: Medium

## User Story
As a visitor, I want to search for posts by keyword, so that I can find content relevant to my interests.

### Acceptance Criteria
- [ ] Any user (authenticated or not) can search using a keyword.
- [ ] Results include posts whose title or description match the keyword, ordered by relevance.
- [ ] The user can filter results by hobby.
- [ ] The user can filter results by post type.
- [ ] If no posts match the search, the system displays a no-results message.
