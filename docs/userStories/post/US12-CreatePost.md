## US-12 — Create Post

## Priority: High

## User Story
As an authenticated user, I want to create a post, so that I can share questions, tips, or achievements with the community.

### Acceptance Criteria
- [ ] The user must provide a title, description, hobby, and post type (Question, Tip, or Achievement).
- [ ] The system must reject submission if no hobby is selected.
- [ ] The system validates all required fields before creating the post.
- [ ] The post is created and the user is redirected to the new post page.
