## US-08 — View Own Profile

## Priority: Medium

## User Story
As an authenticated user, I want to view my own profile, so that I can see how my profile appears to others and access my account settings.

### Acceptance Criteria
- [ ] The user can access their own profile page.
- [ ] The profile displays their username, description, hobbies, and posts.
- [ ] The profile displays an edit button to allow the user to edit their account information.
