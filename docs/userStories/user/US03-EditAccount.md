## US-03 — Edit Account

## Priority: Medium

## User Story
As an authenticated user, I want to edit my account information, so that I can keep my profile up to date.

### Acceptance Criteria
- [ ] The user can edit their username, description, and email.
- [ ] All fields are optional — only the provided fields are updated.
- [ ] The system must reject an email that is already registered to another account.
- [ ] The system must reject a username that is already taken.
- [ ] The system must validate the provided data before saving.
- [ ] The changes are persisted and the user sees the updated profile.
