## US-02 — Login

As a visitor, I want to log in to my account, so that I can access the platform and interact with the community.

### Acceptance Criteria
- [ ] The visitor must provide an email and password.
- [ ] The visitor must have already created an account.
- [ ] The system must reject invalid credentials (email not found or incorrect password).
- [ ] The user is authenticated and redirected to the main page.
