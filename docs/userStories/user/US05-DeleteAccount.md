## US-05 — Delete Account

## Priority: Medium

## User Story
As an authenticated user, I want to delete my account, so that my personal data is removed from the platform.

### Acceptance Criteria
- [ ] The user must confirm the deletion by providing their current password.
- [ ] The system must reject the deletion if the provided password is incorrect.
- [ ] The account is permanently deleted.
- [ ] All posts and comments previously associated with the account are attributed to "Deleted User".
- [ ] All votes cast by the deleted account are permanently removed.
- [ ] The session is terminated and the user is redirected to the home page.
