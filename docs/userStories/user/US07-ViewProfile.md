## US-07 — View Profile

## Priority: Medium

## User Story
As a visitor, I want to view a user's profile, so that I can learn more about them and their interests.

### Acceptance Criteria
- [ ] The visitor can access any user's public profile page.
- [ ] The profile displays the user's username, description, hobbies, and posts.
- [ ] If the profile does not exist, the system displays a not found message.
