## US-11 — Search Hobby

## Priority: High

## User Story
As a visitor, I want to search for a hobby, so that I can find content and people related to my interests.

### Acceptance Criteria
- [ ] The visitor can search for a hobby by name.
- [ ] The system returns hobbies that match the search term.
- [ ] If no hobbies match the search term, the system displays a message indicating no results were found.
