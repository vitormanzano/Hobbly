## US-25 — Review Hobby Suggestion

## Priority: Medium

## User Story
As an admin, I want to approve or reject hobby suggestions submitted by users, so that I can keep the platform's hobby list curated and relevant.

### Acceptance Criteria
- [ ] The admin can view a list of pending hobby suggestions.
- [ ] Each suggestion displays the proposed hobby name and the user who submitted it.
- [ ] The admin can approve a suggestion, making the hobby available on the platform.
- [ ] The admin can reject a suggestion, permanently discarding it.
- [ ] If there are no pending suggestions, the system displays a message indicating the list is empty.
