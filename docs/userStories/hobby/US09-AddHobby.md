## US-09 — Add Hobby

## Priority: High

## User Story
As an authenticated user, I want to add a hobby to my profile with how long I have been practicing it, so that I can connect with people who share the same interests.

### Acceptance Criteria
- [ ] The user can search for a hobby from the existing list.
- [ ] The user must select how long they have been practicing the hobby from a fixed scale: Less than 6 months, Less than 1 year, 1 year, 2 years, 3 years, 4 years, 5+ years, 10+ years.
- [ ] The same hobby cannot be added twice to the same profile.
- [ ] The hobby and practice time are displayed on the user's public profile.
- [ ] If no hobby fits the user's interest, the user can suggest a new one for admin approval.
