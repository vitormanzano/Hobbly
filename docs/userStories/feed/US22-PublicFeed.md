## US-22 — View Public Feed

## Priority: High

## User Story
As a visitor, I want to see the most upvoted posts across all hobbies, so that I can discover popular content without needing an account.

### Acceptance Criteria
- [ ] Any user (authenticated or not) can access the public feed on the home page.
- [ ] Posts are displayed ordered by vote count descending.
- [ ] If the platform has no posts yet, the system displays a message encouraging the user to create an account and share the first post.
