## US-23 — View Personalized Feed

## Priority: High

## User Story
As an authenticated user, I want to see a feed personalized to my hobbies, so that I can find content most relevant to my interests.

### Acceptance Criteria
- [ ] The personalized feed displays a mix of the most upvoted posts from the user's hobbies and from the platform overall, ordered by vote count descending.
- [ ] If the user has no hobbies added, the system falls back to the public feed and suggests adding hobbies to personalize the feed.
- [ ] If the user's hobbies have no posts yet, the system falls back to the public feed.
