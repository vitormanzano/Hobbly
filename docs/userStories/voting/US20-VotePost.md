## US-20 — Vote on Post

## Priority: High

## User Story
As an authenticated user, I want to upvote or downvote a post, so that I can express whether I find the content valuable or not.

### Acceptance Criteria
- [ ] Only authenticated users can vote on a post.
- [ ] A user cannot vote on their own post.
- [ ] Clicking the upvote button registers an upvote and updates the post's vote count.
- [ ] Clicking the downvote button registers a downvote and updates the post's vote count.
- [ ] Clicking the same vote type again removes the vote (toggle) and updates the vote count.
- [ ] Clicking the opposite vote type switches the vote and updates the vote count accordingly.
- [ ] If the post was deleted before the vote is registered, the system displays an error and does not register the vote.
