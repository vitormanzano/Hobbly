## US-24 — Receive Notifications

## Priority: Medium

## User Story
As an authenticated user, I want to receive notifications when others interact with my content, so that I can stay informed about activity on my posts and comments.

### Acceptance Criteria
- [ ] The user receives a notification when another user comments on their post.
- [ ] The user receives a notification when another user replies to their comment.
- [ ] The user receives a notification when another user votes on their post or comment.
- [ ] The notification indicator displays the count of unread notifications.
- [ ] Clicking the notification indicator displays the list of notifications ordered by most recent.
- [ ] If the user has no notifications, the system displays a message indicating there are no notifications yet.
- [ ] The user can mark all notifications as read, clearing the notification count indicator.
